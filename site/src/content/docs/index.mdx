## One repo or many

Single-project usage is the default. Run `momentum init` in any directory and
get a complete workflow — no ecosystem files, no cross-repo state. When you
need to coordinate across multiple related projects, ecosystem mode lets one
agent session orchestrate them together without losing momentum's per-repo
tracking discipline.
