Install momentum, scaffold a project, and run your first phase.

## Install

```bash
npx @avinash-singh-io/momentum init
```

## Next

- Open your agent (Claude Code, Codex, Antigravity) in the project directory.
- Run `/start-project` to scaffold from a clear idea.
- Run `/brainstorm-phase` to plan your next phase.
- Run `/start-phase` to begin implementation.
- Run `/complete-phase` to verify, release, and commit.
